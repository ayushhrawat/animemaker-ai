# AnimeMaker AI

AnimeMaker AI is a web application that allows users to create animated videos using AI technology. The application provides a complete workflow from story creation to final video export with customizable characters, scenes, voiceovers, and music.

## Features

- **Story Input**: Create your anime story with customizable prompts
- **Character Design**: Design unique characters with various customization options
- **Scene Editing**: Create and edit scenes with different backgrounds
- **Voiceover Selection**: Choose from multiple voice options for your characters
- **Music & Effects**: Add background music and sound effects to enhance your video
- **Preview**: Preview your animated video before export
- **Output Style**: Select resolution and quality for final export

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

### Usage

1. Open the application in your browser
2. Create an account or log in
3. Start a new project
4. Follow the step-by-step workflow:
   - Enter your story
   - Design characters
   - Edit scenes
   - Add voiceovers
   - Include music and sound effects
   - Preview your video
   - Export in your preferred format

- React.js
- React Router

## Workflow

1. **Login/Signup**: Access the application with your credentials

4. **Scene Editing**: Design scenes with various backgrounds
5. **Voiceover**: Select voices for your characters
